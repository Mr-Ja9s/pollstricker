# Polling System (Real-Time Voting App)

## Description
A real-time polling system that allows users to create polls and vote dynamically with live updates.

## Features
- Create polls with multiple options
- Real-time voting updates using Socket.io
- REST API for poll management
- Dynamic UI updates

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Socket.io
- React (Frontend)
