# Polling System (Pollstricker)

A real-time polling application that allows users to create polls, vote instantly, and view live results.

---

## Tech Stack
- Node.js
- Express.js
- JavaScript
- HTML, CSS

---

## Features
-  Create polls with multiple options  
-  Vote in real-time  
-  Live result updates  
-  Simple and responsive UI  

---

## Project Structure
poll-project/   → Core logic  
index.js        → Server entry point  
index.html      → Frontend UI  

---

## How to Run
1. Clone the repository  
2. Install dependencies: npm install  
3. Start the server: node index.js
